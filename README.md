# 🎾 Tennis Session Booking Bot

A comprehensive Telegram bot for booking tennis sessions with Google Calendar integration.

## Features

- 📅 View available time slots from Google Calendar
- 🎾 Book tennis sessions with one-time booking
- ✏️ Modify or cancel existing bookings
- 🔔 Automatic coach notifications
- 🗄️ SQLite database for local storage
- 🌐 Google Calendar synchronization
- 📱 User-friendly Telegram interface
